# Análisis de Bienes Muebles Histórico-Artísticos | Madrid 2022


Este proyecto analiza el inventario oficial de bienes muebles histórico-artísticos de la ciudad de Madrid correspondiente al año 2022.

El dataset contiene información detallada sobre 9.561 bienes catalogados, incluyendo tipología, ubicación institucional y descripción.

El objetivo es medir volumen, concentración patrimonial, diversidad tipológica y estructura descriptiva.



# Resultados del análisis

## ¿Cuántos bienes existen en total?

Total inventariado:

**9.561 bienes muebles histórico-artísticos**

Esto refleja un volumen patrimonial amplio y estructurado bajo gestión municipal.



## 2️⃣ ¿Qué tipologías predominan?

Top 5 catalogaciones:

- **PINTURAS** → 1.757 registros  
- **ESTAMPAS** → 845  
- **FUENTES ORNAMENTALES** → 572  
- **FOTOGRAFÍAS** → 434  
- **ESTATUAS Y GRUPOS ESCULTÓRICOS** → 422  

📌 El Top 5 concentra:

**42,15% del inventario total**

Existe especialización clara en pintura y obra gráfica.



## 3️⃣ ¿Dónde se concentran los bienes?

Distribución por subepígrafe:

- **MUSEOS** → 6.100 bienes  
- **PARQUES Y VÍAS PÚBLICAS** → 1.755  
- **DEPENDENCIAS MUNICIPALES** → 1.410  

📌 El 63,8% del patrimonio se encuentra en Museos.



## 4️⃣ ¿Existe concentración patrimonial?

Se calculó el Índice Herfindahl-Hirschman (HHI):

**HHI = 0,0627**

Interpretación:

- Concentración moderada.
- No existe monopolio tipológico absoluto.
- Determinadas categorías (como Pinturas) dominan, pero el sistema mantiene variedad.



## 5️⃣ ¿Cuál es la diversidad de categorías?

Se calculó la entropía del sistema:

**Entropía = 4,78**

Interpretación:

- Alta diversidad estructural.
- Amplia dispersión tipológica.
- Patrimonio relativamente equilibrado entre múltiples categorías.


## 6️⃣ ¿Qué palabras son más frecuentes en las descripciones?

Tras limpieza básica del texto:

Top palabras:

- sobre (3.113)
- altura (1.781)
- madrid (1.728)
- bronce (1.599)
- anchura (1.465)
- madera (1.459)
- óleo (1.421)
- lienzo (1.311)
- forma (1.290)
- dorado (1.179)

Insight:

Las descripciones se centran en:

- Materiales (bronce, madera)
- Técnicas (óleo, lienzo)


# Conclusiones estratégicas

- Inventario amplio y bien estructurado.
- Alta concentración institucional en Museos.
- Concentración tipológica moderada (HHI 0,0627).
- Diversidad significativa (Entropía 4,78).
- Fuerte presencia de pintura dentro del patrimonio.
